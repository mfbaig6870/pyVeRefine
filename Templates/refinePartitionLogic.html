d. Average AFR Bins

This script processes a log file of vehicle performance data using the pandas library. It filters the data based on a specified threshold value and creates partitions for the RPM and MAP values using predefined ranges. The script then groups the data by RPM and MAP bins and calculates the mean value for a certain variable. The resulting data is rounded and reshaped into a grid, which is saved as a CSV file. The purpose of this script is to enable direct analysis of the data by condensing it into a structure resembling a volumetric efficiency table.


e. Average Target AFR Bins

In this script, the log file is loaded into a DataFrame, the data is filtered by the TPS column, keeping only the rows where the value in that column is greater than 10. Partition ranges are defined for RPM and MAP values, and bins are created using these partition ranges. The data is grouped by the RPM and MAP bins, and the mean Target AFR value is calculated for each group. The resulting DataFrame is then reshaped into a grid of MAP vs RPM, with the mean AFR values displayed in each cell. Finally, the resulting grid is saved as a CSV file.


f. Average AFR Delta

This script performs a data manipulation task involving two input DataFrames and one output DataFrame. It uses the pandas library to read the input DataFrames and write to the output DataFrame. The script calculates the percentage difference between the corresponding values of the two input DataFrames, skipping the first row and column which are assumed to contain headers and row labels. If the denominator is zero when calculating the percentage difference, the script outputs 0. The resulting output DataFrame has the same number of rows as the input files and each cell in the output file contains the percentage difference calculated from the corresponding cells in the input files.
