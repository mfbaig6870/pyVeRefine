pyVeRefine.

A public utility to ingest, characterize, and filter large datasets for the refinement of Volumetric Efficiency tables.


1. Upload a Log File and VE Table

(1) Upload a data log file for consumption and analysis and (2) the volumetric efficiency table corresponding to the data log file.


2. Characterize and Filter Log Data

Characterize records in the data log file to identify specific events and filter the dataset to exclude superfluous data from calculation.


3. Apply Bin Partition Logic

Place records from the data log files into collection bins partitioned by engine RPM and manifold pressure, replicating the physical structure of the VE Table.


4. Calculate Air Fuel Ratio Variance

With the data log file records sorted into RPM x MAP partition bins, average the Actual AFR and Target AFR data within the bins, then calculate the variance between the two fields to determine the percentage of correction necessary.


5. Apply Correction to VE Table

Apply the AFR correction to the VE Table.


6. Apply Smoothing Logic to VE Table

Apply a smoothing logic to the revised VE Table to ensure cell to cell transitions fall within an acceptable range for smooth engine operation.
